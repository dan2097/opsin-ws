  • Department of Chemistry

Enter a chemical name into the box then click submit. If the name can be interpreted a depiction, SMILES and CML will be returned.

Updated 30/6/10: Bug fixes, new CIP algorithm and added support for a few more functional classes e.g. diketone, disulfone
Depiction and SMILES courtesy of the CDK. Note that this web interface is primarily for demonstration purposes.
